Contexte :

Une initiative personnelle afin d'améliorer mes connaissances. Ce RPG basique utilise la génération procédurale de niveaux, des cookies, de la programmation orientée objet ... Il n'y a actuellement aucune histoire/scénario (mais ça viendra).
Dans sa version 1, j'utilise le javascript sans librairie complémentaire.
Dans sa version 2, l'ajout de JQuery simplifie grandement la gestion des événements.
L'ensemble du projet est hébergé sur GitHub.

Technologies :

  • HTML 5 (Canvas)
  • CSS 3
  • JavaScript (JQuery)
  • Git
